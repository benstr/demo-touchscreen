Introducing the Hologram Nova

The Nova is an open-sourced USB modem made for IoT Engineers and Makers. Nova is the first solution to make cellular easy for Single Board Computers. Below is a quick demo showcasing the most common uses.


To see data sent from this device to the cloud go to dashboard.hologram.io/?drawer=full from another computer (credentials should have been emailed).


Note: make sure to plug in the Nova USB before clicking any buttons below. Wait until Nove has a solid blue LED and a blinking red LED.

Send Data to the Cloud

Easily send data to the cloud through cellular. Clicking the button below will send a message to our cloud. Go to the online dashboard from another computer dashboard.hologram.io/?drawer=full.

## Example of the code running now

hologram.sendMessage( "Hello Nova" )

Send SMS to Your Phone

Enter your number below to send yourself a text message. In the next step below you'll respond to the message and see it appear.

## Example of the code running now

hologram.sendSMS( "+11234567890" , "Hello Nova" )

Receive Inbound SMS/Data

{{inboundMessage}}

## Example of the code running now

hologram.enableSMS()

hologram.popReceivedSMS()
