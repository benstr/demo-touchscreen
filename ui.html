Introducing the Hologram Nova

The Nova is an open-sourced USB modem made for IoT Engineers and Makers. Below is a quick demo showcasing how easy it is for developers to connect their projects to the web with the Nova. To see data sent from the device to the cloud go to https://www.glitchwebsite.com

Send Data to the Cloud

Easily send data to the cloud through cellular

## Abbreviated python code

hologram.sendMessage("Hello Nova")

If you prefer SMS you can do that too!
